# Pixelpay Wallet

A simple wallet for the [pixelpay](https://github.com/adibfahimi/pixelpay) project.

## Warning!!

Your private key is saved in the `config.json` file. **Do not delete this file.**

For enhanced security, it is highly recommended to move the `config.json` file to a USB device or any other secure storage medium.
